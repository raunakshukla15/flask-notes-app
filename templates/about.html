How it works

This website lets you create a note with an optional deadline. Notes are saved in a local SQLite database on your machine, so they persist after refresh.

UI
HTML + CSS (gradient background + glassmorphism)
Interactivity
Vanilla JavaScript + Fetch API (no reload CRUD)
Backend
Python + Flask (JSON endpoints)
Storage
SQLite (built-in, file: notes.db)

Features

  • Create notes with a timestamp
  • Optional deadline tracking (near / overdue styling)
  • Edit and delete notes instantly
  • Light/Dark theme toggle (saved in the browser)
  • Search/filter your notes
